Frontend Design Document
Product: Dobby

Product Type: Autonomous AI Job-Hunting Agent
Design Philosophy: Minimal • Futuristic • Transparent • Functional

Inspired by the design language of Nothing.

1. Design Philosophy

Dobby’s UI should feel like:

Effortless

Minimal

Focused

Technologically advanced

The interface should communicate:

“The AI is doing the work for you.”

Users should feel in control but not overwhelmed.

2. Core Design Principles
1. Radical Simplicity

Minimal UI elements with strong visual hierarchy.

2. Transparency

System activity must always be visible.

3. Motion with Purpose

Animations should show AI activity and decision making.

4. Data Clarity

Key metrics must be immediately visible.

3. Visual Identity
Color Palette

Primary colors follow a Nothing-style monochrome palette.
